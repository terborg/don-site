# MijnProducten

MijnProducten biedt de inwoner en ondernemer een overzicht van aangevraagde
producten en diensten bij de overheidsorganisatie. Hierin verschijnt
bijvoorbeeld zijn vergunning, zodra deze is goedgekeurd.

import Link from "@docusaurus/Link";
import {
  useCurrentSidebarSiblings,
  filterDocCardListItems,
  useDocById,
} from "@docusaurus/plugin-content-docs/client";
import styles from "@site/src/theme/DocCard/styles.module.css";

export const BouwsteenCard = ({ item }) => {
  const doc = useDocById(item.docId ?? undefined);
  const description = item.description ?? doc?.description;
  return (
    <article
      className="col col--6"
      style={{ marginBottom: "var(--ifm-spacing-vertical)" }}
    >
      <Link
        href={item.href}
        className={`card padding--lg ${styles.cardContainer}`}
      >
        <h3 className={styles.cardTitle}>
          {item.customProps?.icon && (
            <img
              src={item.customProps.icon}
              className={styles.cardIcon}
              style={{ objectFit: "contain" }}
              alt=""
              aria-hidden="true"
            />
          )}{" "}
          {item.label}
        </h3>
        {description && <p className={styles.cardDescription}>{description}</p>}
      </Link>
    </article>
  );
};

export const BouwsteenCardList = () => (
  <section className="row">
    {filterDocCardListItems(useCurrentSidebarSiblings()).map((item, i) => (
      <BouwsteenCard key={i} item={item} />
    ))}
  </section>
);

# Bouwstenen

De services zijn geen losse eilandjes, maar vormen samen een logische keten.
Hieronder ziet u de services in de volgorde van een typische klantreis: van het
eerste signaal tot het eindresultaat.

## MijnTaken

De actie. Soms wordt er iets van de inwoner verwacht. Via MijnTaken handelt de
inwoner specifieke acties af, zoals het betalen van leges, het aanvullen van
gegevens of het bevestigen van een afspraak. Zodra de taak is afgerond,
verdwijnt deze uit de actielijst.

## MijnZaken

Het proces. Zodra een aanvraag loopt, wil de inwoner weten waar hij aan toe is.
MijnZaken is de 'track & trace' van de overheid. Het toont de status van lopende
processen, de verwachte doorlooptijd en de stappen die al zijn genomen.

## MijnProducten

Het resultaat. Als een proces succesvol is afgerond, levert dit vaak iets
tastbaars op. MijnProducten toont een overzicht van de actieve rechten en
bezittingen van de inwoner, zoals een verleende parkeervergunning, een paspoort
of een huurcontract.

## MijnContactmomenten

De historie. Inwoners willen niet alleen de status zien, maar ook wat er is
besproken. MijnContactmomenten fungeert als het geheugen van de dienstverlening.
Het toont een integrale tijdlijn van alle contacten (mail, telefoon, balie) die
bij een specifieke zaak horen.

## MijnActies en MijnProfiel

Proactief en beheer. Met MijnActies (niet te verwarren met taken) biedt u
proactieve tips, zoals het tijdig verlengen van een paspoort. In MijnProfiel
beheert de inwoner zijn voorkeuren, zoals de manier waarop hij benaderd wil
worden.

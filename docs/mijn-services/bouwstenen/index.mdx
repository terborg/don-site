import Link from "@docusaurus/Link";
import {
  useCurrentSidebarSiblings,
  filterDocCardListItems,
  useDocById,
} from "@docusaurus/plugin-content-docs/client";
import styles from "@site/src/theme/DocCard/styles.module.css";

export const BouwsteenCard = ({ item }) => {
  const doc = useDocById(item.docId ?? undefined);
  const description = item.description ?? doc?.description;
  return (
    <article
      className="col col--6"
      style={{ marginBottom: "var(--ifm-spacing-vertical)" }}
    >
      <Link
        href={item.href}
        className={`card padding--lg ${styles.cardContainer}`}
      >
        <h3 className={styles.cardTitle}>
          {item.customProps?.icon && (
            <img
              src={item.customProps.icon}
              className={styles.cardIcon}
              style={{ objectFit: "contain" }}
              alt=""
              aria-hidden="true"
            />
          )}{" "}
          {item.label}
        </h3>
        {description && <p className={styles.cardDescription}>{description}</p>}
      </Link>
    </article>
  );
};

export const BouwsteenCardList = () => (
  <section className="row">
    {filterDocCardListItems(useCurrentSidebarSiblings()).map((item, i) => (
      <BouwsteenCard key={i} item={item} />
    ))}
  </section>
);

# Bouwstenen

De services zijn geen losse eilandjes, maar vormen samen een logische keten.
Hieronder ziet u eerst de bouwstenen uit de pocketgids MijnServices in
zakformaat. Daarna volgen aanvullende bouwstenen die in deze documentatie al in
uitwerking zijn.

## MijnZaken

MijnZaken is een track & trace-service waarmee inwoners en ondernemers hun
lopende zaken bij uw organisatie kunnen volgen.

## MijnTaken

MijnTaken is een actie die de inwoner moet uitvoeren en die meestal vanuit het
overheidsportaal plaatsvindt.

## MijnBerichten

MijnBerichten biedt de mogelijkheid om informatieve berichten of meldingen te
versturen aan inwoners en ondernemers. Bijvoorbeeld over een taak die nog
uitgevoerd moet worden of over een genomen besluit. Dit werkt hetzelfde als de
berichtenbox van MijnOverheid.

## MijnContactmomenten

Deze service geeft een overzicht van contactmomenten tussen gemeenten en inwoner
of ondernemer over een specifieke zaak.

## MijnProfiel

Via deze service kunnen onder andere persoonlijke voorkeuren voor contact met
overheidsorganisaties worden beheerd. Denk bijvoorbeeld aan kanaalvoorkeuren van
de inwoner of ondernemer. Willen ze digitaal of per brief op de hoogte worden
gehouden?

## MijnProducten

MijnProducten biedt de inwoner en ondernemer een overzicht van aangevraagde
producten en diensten bij de overheidsorganisatie. Hierin verschijnt
bijvoorbeeld zijn vergunning, zodra deze is goedgekeurd.

## MijnActies

MijnActies toont de inwoner of ondernemer mogelijke acties die hij kan
uitvoeren, zoals het verlengen van zijn parkeervergunning. Hiermee kunt u
proactief bepaalde dienstverlening onder de aandacht brengen bij inwoners en
ondernemers.

## Notificatieservice

Deze service zorgt ervoor dat inwoners en ondernemers een notificatie ontvangen
als er bijvoorbeeld een nieuw bericht of een nieuwe taak voor ze klaarstaat, of
als er iets is veranderd in de status van hun aanvraag.

## Aanvullende bouwstenen

Naast de bouwstenen uit de pocketgids werken we in deze documentatie ook al aan
aanvullende services, zoals MijnAgenda en MijnBetaaloverzicht.

# Notificatieservice

Deze service zorgt ervoor dat inwoners en ondernemers een notificatie ontvangen
als er bijvoorbeeld een nieuw bericht of een nieuwe taak voor ze klaarstaat, of
als er iets is veranderd in de status van hun aanvraag. Met deze service
integreert u vereisten van de Wet modernisering elektronisch bestuurlijk verkeer
(Wmebv).
